🎭 The Meme Factory
The Meme Factory is a dynamic web application that allows users to browse trending meme templates and visualize their own captions in real-time. Built with a focus on responsive design and seamless API integration, it brings the "Pinterest" aesthetic to the world of memes.

🎯 Project Purpose
The goal of this project is to create an interactive gallery where users can:

Explore a vast library of meme templates via the Imgflip API.

Search and filter templates by category or popularity.

Live-edit memes by overlaying custom text on selected images.

✨ Planned Features
Masonry Grid Layout: A responsive, multi-column layout to handle various image aspect ratios elegantly.

Real-time Meme Creator: Input fields for "Top Text" and "Bottom Text" that update a preview image instantly.

Search & Filtering: A functional search bar and category tabs (e.g., Trending, Classic) for quick navigation.

Share Functionality: A "Copy Link" button to easily share template URLs to the clipboard.

UX Extras: A "Back to Top" navigation button and smooth hover effects.

🛠️ Technologies Involved
Frontend: React.js / Tailwind CSS

API: Imgflip Public API

Deployment: Vercel / GitHub Pages

Icons: Lucide-React
